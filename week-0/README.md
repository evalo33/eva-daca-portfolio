# Week 0 – Python seadistus

## Mis see projekt oli?
Selle nädala eesmärk oli seadistada Pythoni arenduskeskkond ning valmistada ette kõik vajalikud tööriistad DACA programmi jaoks.

## Mida ma tegin?
- Paigaldasin Pythoni ja VS Code'i.
- Lõin virtuaalkeskkonna (venv).
- Paigaldasin vajalikud paketid (pandas, plotly, supabase jne).
- Testisin Supabase ühendust.
- Lõin esimese Jupyter Notebooki.
- Seadistasin GitHubi portfoolio.

## Mida see näitab / milline oli mõju?
Arenduskeskkond on valmis järgnevate nädalate SQL-i, Pythoni ja andmeanalüüsi ülesannete tegemiseks.
