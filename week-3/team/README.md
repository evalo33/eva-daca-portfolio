# Week 3 – Meeskonnatöö

## Eesmärk

Week 3 eesmärk oli õppida ühendama andmeid mitmest tabelist SQL JOIN käskude abil ning teha UrbanStyle andmete põhjal ärilisi järeldusi.

## Meeskonna tulemused

- Kasutati INNER JOIN ja LEFT JOIN päringuid.
- Leiti kliendid, kellel puuduvad ostud.
- Analüüsiti toodete müüki ja laoseisu.
- Võrreldi müügikanaleid ning klientide ostukäitumist.

## Minu panus

Minu ülesanne oli:
- analüüsida müümata tooteid ja inventuuri;
- võrrelda müügikanalite tulemuslikkust;
- koostada SQL päringud kolme tabeli JOIN-idega;
- teha tulemuste põhjal soovitused Anna Metsale.

Esitlesin oma tulemusi meeskonna ühises esituses.
