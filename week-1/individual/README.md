# Week 1 – SQL põhitõed

## Mis see projekt oli?
Selle nädala eesmärk oli õppida SQL-i põhilisi päringuid UrbanStyle andmebaasis ning analüüsida müügiandmeid.

## Mida ma tegin?
- Importisin UrbanStyle andmed Supabase'i.
- Kasutasin SELECT ja FROM päringuid.
- Filtreerisin andmeid WHERE käsuga.
- Sorteerisin tulemusi ORDER BY abil.
- Kasutasin COUNT ja DISTINCT funktsioone.
- Otsisin duplikaate GROUP BY ja HAVING abil.

## Mida see näitab / milline oli mõju?
Õppisin lugema ja analüüsima andmeid SQL päringute abil ning leidma duplikaate, NULL-väärtusi ja olulisi mustreid.

## AI kasutamine
Kasutasin ChatGPT abi SQL päringute selgitamiseks, vigade parandamiseks ja portfoolio vormistamiseks.
